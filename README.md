## What is FLAMES?

FLAMES is a classic name game that's been passed around playgrounds and classrooms for generations. Each letter stands for a possible relationship between two people:

| Letter | Meaning        |
|--------|----------------|
| 🤝 **F** | Friends        |
| 💕 **L** | Lovers         |
| ✨ **A** | Admires        |
| 💍 **M** | Marriage       |
| ⚔️ **E** | Enemies        |
| 🤫 **S** | Secret Lovers  |

Enter two names, and the game reveals which one describes your connection!

## How to Use

1. Open the `flames.html` file in any web browser (Chrome, Firefox, Safari, Edge — all work great)
2. Type the **first name** into the top box
3. Type the **second name** into the bottom box
4. Hit the **"Reveal Destiny"** button
5. See your result! 🎉

That's it — no instructions manual needed.

---

## How It Works

The game compares the two names and looks for letters they **don't share**. Those leftover letters are counted up, and that count points to one of the six FLAMES outcomes.

Here's the full picture in plain steps:

1. Go through the first name, letter by letter
2. Keep only the letters that **don't appear** in the second name
3. Do the same for the second name — keep letters not found in the first
4. Put all those kept letters together and count them
5. That total number lands on one of the six FLAMES results

> **Example:** "Alice" and "Bob" — the letters they don't share give a count that maps to **LOVERS** 💕

## Getting Started

### Easiest way — just open the file
1. Download or copy the project to your computer
2. Find the file called `flames.html`
3. Double-click it — it opens straight in your browser, ready to go

### Run it from a local server (optional)
If you'd rather serve it up properly, open your terminal and run:
```bash
python -m http.server 8000
```
Then go to `http://localhost:8000/flames.html` in your browser.

### Use it online
If it's been published to GitHub Pages, you can visit it at:
